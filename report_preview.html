Abstract

This study presents a comprehensive deep learning system for automated classification of lung histopathology images into three distinct categories: Adenocarcinoma (ACA), Normal tissue (N), and Squamous Cell Carcinoma (SCC). The proposed system employs a multi-architecture approach combining three pre-trained Convolutional Neural Networks (CNNs) - DenseNet121, ResNet50, and VGG16 - enhanced with Squeeze-and-Excitation (SE) attention mechanisms and multi-head channel attention. A genetic algorithm-based feature selection process optimizes the extracted features, which are then classified using an ensemble of traditional machine learning algorithms including K-Nearest Neighbors (KNN), Support Vector Machine (SVM), and Random Forest. The final prediction is obtained through majority voting fusion. Experimental results demonstrate competitive performance with ensemble accuracy reaching 98.70%, representing a robust alternative to single-architecture approaches.
